# Quorum Education - Site Web

Site web institutionnel pour Quorum Education, spécialisé dans l'acquisition étudiante basée sur la data.

## Structure

- `index.html` - Structure HTML principale avec les sections Hero et Méthode
- `styles.css` - Styles CSS avec design strict (angles droits uniquement)

## Design

### Couleurs
- Bleu nuit : `#0A1326`
- Gris structure : `#D3D9E2`
- Blanc pur : `#FFFFFF`

### Typographies
- Inter (Google Fonts)
- Neue Haas (fallback)
- Graphik (fallback)

### Principes de design
- **Angles droits uniquement** - Aucun élément arrondi
- Formes géométriques carrées
- Grilles strictes 3×3
- Design institutionnel et sobre

## Utilisation

Ouvrir `index.html` dans un navigateur web moderne pour visualiser le site.

## Sections

### Section 1 - Hero
- Bandeau introductif
- Titre principal
- Sous-titre
- Preuves chiffrées (3 éléments)
- CTA "Parler à un expert"

### Section 2 - Méthode
- Titre de section
- Grille 3×3 avec 3 cartes méthodologiques
- Encart de confidentialité discret
